 ## 숫자야구게임
  #### 프로젝트 설명
   컴퓨터가 임의로 지정한 중복되지 않는 3자리 수를 맞추는 게임입니다. <br />
   <게임 방법> <br />
   1. 중복되지 않는 3개의 숫자를 입력합니다. <br />
   2. 규칙에 따라 스트라이크, 아웃, 볼로 표시됩니다. <br />
   3. 3스트라이크까지 게임을 반복하여 계속 진행합니다. <br />
   4. 몇 번만에 맞췄는지를 알려준 후 게임이 종료됩니다.
   
  #### 프로젝트 정보
   제작 기간 : 2주 <br />
   참여 인원 : 1명 <br />
   사용 기술 : C++
